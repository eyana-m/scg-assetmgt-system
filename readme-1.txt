PBI Motolite Asset Management System
Created December 2014 - February 2015

How to use the system.
1. Install XAMPP in your computer
https://www.apachefriends.org/download.html
2. Clone the repository in the /xampp/htdocs or /www/var/ folder
SSH: git@github.com:eyana-m/scg-assetmgt-prototype.git
HTTPS: https://github.com/eyana-m/scg-assetmgt-prototype.git

or

2. Copy paste this whole folder in the /xampp/htdocs (Windows/Mac) or /www/var/ (Linux) folder

3. Go to your browser and type localhost/phpmyadmin

4. Import the sql file with the latest date (Sql files are found in the db_dump folder)

4. Admin Account: 
User Name = admin@gmail.com
Password = adminadmin
